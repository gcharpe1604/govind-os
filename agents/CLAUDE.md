# Claude Capability Overlay

Load [../AGENTS.md](../AGENTS.md) as the policy. This optional overlay contains only Claude-specific reasoning guidance.

- Use broad context only when the decision spans multiple components; otherwise keep context narrow.
- For consequential trade-offs, test the proposal through contrarian, first-principles, upside, outsider, and execution lenses, then make one recommendation.
- Separate assumptions, evidence, risks, and unknowns.
- Stop analysis when additional reading is unlikely to change the decision.

Avoid agreement bias, false precision, framework creation without repeated need, and using a large context window as a reason to load unrelated files.
