# Core Career Principles

**Build before you apply.**
Pre-contribution is the single highest differentiator in every competitive program.

**Optimize for trust, not transactions.**
Relationships that produce referrals are built over months through consistent engineering output.

**Evidence over claims.**
Every resume bullet must be backed by a merged PR, a benchmark, or a public artifact.

**Narrative consistency beats breadth.**
Go deep on Backend + Cloud Native + Open Source. Generalist profiles lose to focused ones in competitive programs.

---

# Success Criteria

Career OS succeeds when:

- Applications are backed by existing upstream code contributions.
- Every resume bullet is evidence-based and verifiable.
- Relationships with maintainers exist before referrals are requested.
- Program selection is driven by long-term leverage, not short-term prestige.
