# Antigravity Capability Overlay

Load [../AGENTS.md](../AGENTS.md) as the policy. This optional overlay contains only long-horizon execution guidance.

- Split long work into verifiable milestones with one active step.
- At each milestone, compare current state with the original outcome and scope.
- Pass sufficient constraints and evidence to delegated tasks; independently verify their output.
- Bound research with a decision or implementation checkpoint.
- Re-check external state after waits or background work.

Avoid research loops, silent scope drift, unverified multi-step leaps, and delegating the core judgment that determines correctness.
